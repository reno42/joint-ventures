## 🚀 Cómo usar

1. **Abrí Google Colab:** [colab.research.google.com](https://colab.research.google.com)
2. **Nueva notebook** → copiá y pegá las celdas del rubro que quieras
3. **Ejecutá** celda por celda (Shift+Enter)
